PRETTY CHECKBOX

No more boring old fashioned checkboxes. New scalable CSS3 pretty checkbox and radio buttons. - No Javascript!

Github Download

Installation

Step 1: Download from bower or node 

> bower install pretty-checkbox

Step 2: Add pretty.css file in your html or import pretty.scss file in your css/sass file

@import '../bower_modules/pretty-checkbox/src/pretty.scss';

Step 3: Add the mark up in your file. Can be used with Bootstrap, Foundation frameworks.

<div class="pretty">
  <input type="checkbox"/> 
  <label><i class="mdi mdi-close"></i>Check me</label> 
</div>

Note: mdi mdi-close is from material design icons library. If you use fontawesome ,then it will be fa fa-close

Sass variables

Default class name

$pretty-class-name:notsopretty;
// <div class="nosopretty inline circle">

Box border radius

$pretty-border-radius:2px !default;

Note: Import the pretty.scss file after declaring the variables
